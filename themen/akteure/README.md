# Akteure / Handles

## Zweck

Neutrale Sammlung von beobachteten Handles/Accounts/Identifikatoren.

Regeln:

- Keine privaten Adressen/Telefonnummern/IDs veröffentlichen.
- Keine unbelegten Zuordnungen zu realen Personen.
- Wenn ein Handle erwähnt wird: Kontext (Plattform, Datum, Interaktion) dokumentieren.

## Strukturvorschlag

- Plattform
- Handle / Identifier
- Beobachtung (Datum/Quelle)
- Verweise auf Belege/Indizien
