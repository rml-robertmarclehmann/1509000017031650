# Rechtliches / Kommunikation

## Zweck

Ablage für formale Zusammenfassungen, Kommunikationsverlauf, Kontaktpunkte und Entwürfe.

Hinweise:

- Neutral schreiben.
- Datum/Uhrzeit, Kanal, Ansprechpartner, Ticket-/Aktenzeichen dokumentieren.
- Keine Zugangsdaten oder sensiblen Identifikatoren ungeschützt ablegen.

## Strukturvorschlag

- Zusammenfassungen (1–2 Seiten)
- Kommunikationslog
- Ansprechpartner
- Vorlagen

## Verweise

- Belege: `belege/rechtliches/...`
- Indizien: `indizien/rechtliches/...`
