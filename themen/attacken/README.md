# Attacken / Angriffsarten

## Zweck

Sammlung und Kategorisierung von Angriffsarten (technisch/sozial) inkl. Indikatoren, Auswirkungen und Gegenmaßnahmen.

## Strukturvorschlag

- Beobachtungen (neutral)
- Hypothesen (klar gekennzeichnet)
- Indikatoren / IOCs
- Reproduktionsschritte (wenn möglich)
- Verweise auf Belege/Indizien

## Verweise

- Belege: `belege/attacken/...`
- Indizien: `indizien/attacken/...`
