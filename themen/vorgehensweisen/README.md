# Vorgehensweisen / Taktiken

## Zweck

Dokumentation von Vorgehensweisen/Taktiken. Wenn keine belastbaren Belege vorliegen, bitte als **Hypothese** kennzeichnen.

## Strukturvorschlag

- Beobachtungen
- Hypothesen (klar gekennzeichnet)
- Wiederkehrende Muster
- Trigger/Anlässe (falls erkennbar)
- Verweise auf Belege/Indizien

## Verweise

- Belege: `belege/vorgehensweisen/...`
- Indizien: `indizien/vorgehensweisen/...`
