# Technische Analyse

## Zweck

Technische Befunde, Logs, System-/Netzwerkindikatoren und Reproduktionsschritte.

## Strukturvorschlag

- Systemumgebung (OS, Gerät, Netzwerk)
- Beobachtete Symptome
- Zeitlicher Kontext (Verweis auf `timeline/`)
- Artefakte/Logs (Belege)
- Hypothesen und Gegenprüfungen

## Verweise

- Belege: `belege/technische-analyse/...`
- Indizien: `indizien/technische-analyse/...`
